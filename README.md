# 📦 Plug In — Peripheral Ecommerce

**Plug In** is a modern ecommerce platform focused on selling tech peripherals such as keyboards, headsets, monitors, mice, and more.  
The project aims to deliver a complete full-stack solution built with a **React + TypeScript frontend**, a **Spring Boot backend**, and an **SQL database**.

---

## 🚀 Demo

- **Live Site:** [https://plug-in-ecommerce.vercel.app]

## 💻 Frontend

- ⚛️ React
- 🟦 TypeScript
- 🎨 TailwindCSS
- 🎞️ GSAP (scroll animations)
- 🔀 React Router
- ⚡ Vite
- 📦 Zustand (State Management)

## ⚙️ Backend

- ☕ Java 17+
- 🌱 Spring Boot
- 🗄️ SQL (PostgreSQL)
- 🔗 Spring Web
- 📚 Spring Data JPA

## ☁️ Deployment

- 🔼 Vercel (Frontend)
- ☁️ Render (Backend)
- 🐘 Neon (Database)
